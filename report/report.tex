\section{序論}
\subsection{はじめに}
本研究では、棒状分子液晶5CB,T15の混合系に対してさらにバナナ状分子液晶P8を濃度比を変えて混合し、相転移や相分離の様子を探った。
また、この系の扱いや動的光散乱(DLS)実験について一定の知見を得た。
\subsection{動機と先行研究}
5CB,T15は図のような棒状の分子である。一方P8(P-8-O-PIMB)は図のような湾曲した構造をもつ分子である。
\begin{description}
\item[5CB] \includegraphics[width=4.5cm]{./5CB.eps}
\item[T15] \includegraphics[width=4.5cm]{./T15.eps}
\item[P8] \includegraphics[width=12cm]{./P8.eps}
\end{description}
5CB,T15はI$\rightarrow$N$\rightarrow$Cry,P8はI$\rightarrow$B4なる相転移をすることが知られている。
これらの混合系を冷却すると,棒状分子及びバナナ状分子がそれぞれの相秩序を形成しようとするはたらきが競合する。
既に5CB/P8混合系については山崎(2007),T15/P8混合系については高田(2008)及び藤本(2008)によって研究がなされている。
本研究では5CB,T15とP8の3成分系を観察した。
\section{理論}
\subsection{動的光散乱(DLS)}
動的光散乱は,試料にレーザー光を照射し散乱光強度から試料のゆらぎの時間/空間scaleを知る実験手法である。
\subsubsection{時間自己相関関数}
物理量$X(t)$のゆらぎにおいて,ある時刻の物理量がどの程度の時間幅の影響を有するかを表す指標として時間自己相関関数が用いられる:
\[
C(t) = <X(t')X(t'+t)> = \frac{1}{T}\int_0^TX(t')X(t'+t)dt'
\]
ここに$<X(t')X(t'+t)>$は時間平均または統計平均である。
\subsubsection{時間自己相関関数と緩和時間}
液晶の揺らぎは多くの場合Gauss過程であり、時間自己相関関数は次式に従うことが知られている:
\[
C(t) = y_0 + A * \exp(-(x/\tau)^ \beta)
\]
ここに$\tau$は散乱の減衰の特徴的時間である緩和時間であり,$\beta$はstretch係数と呼ばれる。
\subsubsection{緩和時間と拡散mode}
拡散方程式に由来する系においては緩和時間は散乱ベクトルの2乗に反比例し拡散定数$D$を用いて
\[
\tau=\frac{1}{Dq^2}
\]
のようになることが知られており,これを利用するのは拡散現象の解析では一般的な手法である。
本研究でも緩和時間の波数依存を調べ,系が拡散modeに従っているかどうかを検証する。

系が拡散modeに従わない場合,系の相分離などの構造の空間スケールについての知見を得ることができる。
一方,拡散modeにおいてはさらに拡散定数を決定することで試料の粘性についての知見を得ることができるが、本研究ではそこまで踏み込まない。
\section{実験}
\subsection{試料の調製}
棒状分子液晶5CB,T15を39.9\%:60.1\%の質量比で混合した試料を調製し,以下これを棒状液晶Aとする。
I$\rightarrow$N転移温度は5CBで30℃前後,T15で240℃前後,棒状液晶Aでは160℃前後である。
棒状液晶Aにバナナ状分子液晶P8を混合し,以下P8の質量比をP8濃度とする。
調製した試料のP8濃度は0.0\%,4.8\%,10.3\%,11.0\%,12.9\%,15.1\%,20.3\%,40.3\%である。
本研究では一つの試料に対して複数のcellを製作し比較した。
さらに後述のように日によって濃度比が変わっている可能性を考慮し,経過日数による比較を行った。
\subsection{cellの製作}
cellの製作は次のように行った。
\begin{enumerate}
\item 厚さ1mmのガラスからを小(10mm x 10mm), 大(10mm x 15mm)のガラス片を切り出し,洗剤及び純水を用いて超音波洗浄する。
\item ドライヤーで乾燥させたガラス片にポリイミドを滴下しすぐに分速4000回転程度で回転させてポリイミド薄膜を作る(スピンコーティング)。ポリイミドは冷暗所保管し、常温にしてから使用する。
\item 180℃で30分程度加熱する。シャーレに入れている場合は、シャーレを急冷しないよう注意する。
\item ベルベットで5回ほどガラス片を一方向に擦り、ポリイミドを水平配向させる。
\item 厚さ6$\mu$mのフィルムをスペーサーとして大小1対のガラス片を配向を合わせて対向させ、クリップで挟む。
\item 試料を吸い取らせる部分を残して接着剤を塗布する。ガラス片が曲がるのを防ぐため、クリップで挟んでいる部分にも接着剤を塗りできるだけ均一になるようにする。
\item 60℃で30分ほど加熱するしcellの固定を確認する。
\item 分光器により厚さを測定する。本研究で製作したcellの厚さは6-10$\mu$m程度となった。
\item 針を使い,N相またはI相の温度域まで温めたcell内に分子を浸透させる。
\item 接着剤を用いてcellを密封し再び加熱する.
\end{enumerate}
cellの密封は第70日から行った。さらに第77日以降,より密封性の高い強力な接着剤を用いた。
強力な接着剤の固形化温度は130℃-150℃程度である。常温から100℃程度まで加温するにつれいったん粘性が下がりcell内に浸透しやすくなるので注意を要する。
\subsection{相系列の検討}
まず濃度ごとの相系列を特定するため,偏光顕微鏡とデジタルカメラを用い、各試料をI相から冷却して観察した。
各相の特徴的な様子を後に掲げる。
\subsection{DLS}
製作した各cellに対し、60℃-160℃の温度域においてDLS実験を行った。

\section{課題}
今後精度のよい測定を行うにあたり、実験について得た知見と対策を述べる。
\subsection{濃度比の精度}
cellを高温に保つ時間が長いと成分が蒸発し系の濃度比が調製時と変わるという可能性を指摘した。
対策として考えられるものを挙げる。
\begin{description}
\item[cellの密封]
本研究においても接着剤による密封処理を施したが、より効果的な対策が必要である。
cellを密封した場合の問題として、気体相の成分の圧力により転移の様子が変わりうることが挙げられる。
\item[成分蒸気中に浸す]
どの成分が気化しているか明らかな場合は、定圧に保った飽和蒸気中にcellを浸して実験を行うことが可能であると考えられるが、装置製作のコストを要する。
\item[cellの作り捨て]
上記の対策がとれない場合、技術的に最も現実的なのは、製作したcellには数時間程度の寿命があると見切りをつけcellを多数製作しておくことである。
これもcellの材料及び試料のコストを要する。
\item[別の系として測定する]
問題は濃度比が既知のものから未知のものになる点にある。
異なる既知の濃度比のデータが大量にあれば、成分が蒸発したとしても互いに参照しながら濃度比を既知に戻し有効なデータとして解析できる可能性がある。
\end{description}

\subsection{温度測定の精度}
本研究の装置系に特有の課題であるが、DLS装置において、入射光に対する試料の角度を変えると温度調節が僅かに上下する現象を確認した。
原因と対策として考えられるものを挙げる。
\begin{description}
\item[HEPA filter]
空気中の埃を除去するHEPA filterにより気流が生まれ,試料の角度によって電気炉の温度調節が狂う可能性がある。
対策としてHEPA filterのスイッチを切ることが考えられるが、この場合暗室内の埃が実験に影響を与えないことを保証する必要がある。
\item[熱電対]
熱電対が固定されていないため、試料の角度によって電気炉の温度調節が狂う可能性がある。
対策として熱電対を固定することが考えられる。
\end{description}
